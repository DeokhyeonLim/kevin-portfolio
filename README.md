## Deploy on GitHub Pages

This repository is intended to be published as a GitHub Pages project site.

1. Create or use the repository named `kevin-portfolio`
2. Keep `index.html` at the repository root
3. Keep the `assets/` folder beside `index.html`
4. Go to **Settings → Pages**
5. Set **Source** to `Deploy from a branch`
6. Select the `main` branch and `/ (root)` folder
7. Save the settings
8. Open: https://deokhyeonlim.github.io/kevin-portfolio/
