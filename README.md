# Deokhyeon (Kevin) Lim — Portfolio

A static personal portfolio website for **Deokhyeon (Kevin) Lim**, an IT professional focused on cloud infrastructure, automation, data analytics, cybersecurity, and technical teaching.

## Live Site

- GitHub: https://github.com/DeokhyeonLim

├── assets/

## Sections

- Hero — animated typing intro
- About — bio, photo, quick links
- Education — Flinders University and Kwangwoon University
- Skills — systems, cloud, security, programming, architecture, and leadership
- Projects — academic and professional projects with pagination
- Certifications — swiper carousel of professional credentials
- Awards — academic and volunteer recognition
- Media — embedded professional interview video
- Contact — location, email, education, LinkedIn, and GitHub

## Tech Stack

Bootstrap 5, Bootstrap Icons, AOS, Swiper, Particles.js, Typed.js, HTML, CSS, and JavaScript.

## Deploy on GitHub Pages

This repository is intended to be published as a GitHub Pages user site.

1. Create or use the repository named `deokhyeonlim.github.io`
2. Keep `index.html` at the repository root
3. Keep the `assets/` folder beside `index.html`
4. Commit and push to the `main` branch
5. Open: https://deokhyeonlim.github.io
